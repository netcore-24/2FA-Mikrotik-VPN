- Основные параметры
- Безопасность
- Уведомления администратора

### 2. Настройки Telegram Bot
- Основные параметры
- Тексты сообщений
- Визуальное оформление
- Логика работы

### 3. Настройки MikroTik
- Подключение
- User Manager
- Firewall правила
- Мониторинг

### 4. Настройки пользователей
- VPN параметры
- Уведомления
- Ограничения
- Статусы и разрешения

### 5. Настройки форм и полей
- Регистрация
- Веб-интерфейс

### 6. Настройки логирования и мониторинга
- Логирование
- Мониторинг

### 7. Настройки базы данных
- Резервное копирование
- Очистка

### 8. Настройки веб-интерфейса
- Внешний вид
- Функциональность
