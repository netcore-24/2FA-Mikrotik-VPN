# Управление базой данных

Документация по управлению базой данных SQLite из веб-интерфейса.

## Скачивание базы данных

### Доступ
Раздел "Настройки" -> "База данных" -> "Скачать резервную копию"

### Функции
- Создание резервной копии перед скачиванием
- Скачивание файла .db с текущей датой и временем в имени
- Опциональная компрессия в .zip или .tar.gz
- Логирование операции в журнал аудита

### Безопасность
- Доступ только для супер-администраторов
- Проверка прав доступа перед операцией

## Загрузка базы данных

### Доступ
Раздел "Настройки" -> "База данных" -> "Загрузить резервную копию"

### Процесс
1. Выбор файла .db или архива (.zip, .tar.gz)
2. Проверка формата файла
3. Проверка версии базы данных (совместимость)
4. Создание резервной копии текущей базы
5. Предупреждение о замене текущей базы
6. Подтверждение операции
7. Валидация целостности загруженной базы
8. Замена текущей базы данных
9. Опциональный перезапуск сервисов
10. Логирование операции в журнал аудита

### Безопасность
- Доступ только для супер-администраторов
- Проверка целостности файла перед загрузкой
- Автоматическое создание резервных копий перед операциями
- Валидация версии базы данных

## Информация о базе данных

### Отображаемая информация
- Размер файла базы данных
- Версия схемы базы данных
- Количество записей в каждой таблице
- Дата последнего резервного копирования
- Дата последнего изменения
- Статистика использования дискового пространства

### Доступ
Раздел "Настройки" -> "База данных" -> "Информация"

## Автоматические резервные копии

### Настройка
- Интервал резервного копирования (часы, дни)
- Время начала резервного копирования
- Хранение резервных копий (количество, дни)
- Путь для сохранения резервных копий
- Компрессия резервных копий
- Уведомления об успехе/ошибках

### Доступ
Раздел "Настройки" -> "База данных" -> "Автоматические резервные копии"
