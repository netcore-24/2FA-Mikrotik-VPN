# Алгоритм работы системы

Подробное описание алгоритма работы системы двухфакторной аутентификации.

## Основные этапы

### 1. Регистрация пользователя
Пользователь регистрируется через Telegram бот, администратор проверяет и одобряет регистрацию.

### 2. Запрос подключения к VPN
Пользователь запрашивает подключение через бот, система активирует пользователя в MikroTik User Manager.

### 3. Подключение и подтверждение
Пользователь подключается к VPN, система обнаруживает подключение и запрашивает подтверждение в Telegram.

### 4. Активация правил firewall
После подтверждения включается правило firewall для пользователя, запускается таймер напоминания.

### 5. Напоминание о продолжении работы
Через заданное время система отправляет напоминание, пользователь может продлить или завершить сессию.

Подробное описание каждого этапа см. в `SYSTEM_WORKFLOW.md`
