# Мастер настройки (Setup Wizard)

Документация по мастеру настройки системы.

## Описание

Мастер настройки - это интерактивный инструмент, который помогает администратору настроить систему после первого развертывания. Он предоставляет пошаговое руководство с подсказками и инструкциями на каждом этапе.

## Режимы работы

### Автоматический запуск
- Мастер автоматически запускается при первом входе в систему, если настройка не завершена
- Определяется по наличию созданного администратора и основных настроек

### Ручной запуск
- Мастер можно запустить в любой момент из меню "Настройки" -> "Мастер настройки"
- Полезно для полной переконфигурации системы или добавления новых настроек

### Перезапуск
- Возможность начать настройку заново с любого шага
- Позволяет изменить ранее настроенные параметры

## Шаги мастера настройки

### Шаг 1: Добро пожаловать
Приветственное сообщение с описанием процесса настройки.

### Шаг 2: Основная информация
- Название системы
- Язык интерфейса
- Часовой пояс
- Email администратора

### Шаг 3: Безопасность
- Генерация секретного ключа
- Настройка JWT токенов
- Создание первого администратора

### Шаг 4: Telegram Bot
- Инструкции по созданию бота
- Настройка Telegram Bot Token
- Тест подключения

### Шаг 5: MikroTik Router
- Настройка подключения к роутеру
- Инструкции по настройке доступа
- Тест подключения
- Настройка параметров User Manager и Firewall

### Шаг 6: Уведомления
- Настройка уведомлений администратору
- Telegram ID и Email
- Типы уведомлений

### Шаг 7: Дополнительные настройки (опционально)
- Настройка веб-интерфейса
- Настройка логирования
- Настройка резервного копирования

### Шаг 8: Проверка и завершение
- Сводка всех настроек
- Проверка готовности системы
- Завершение настройки

## Функции

- **Прогресс настройки**: Индикатор текущего шага (X из Y)
- **Навигация**: Кнопки "Назад", "Далее", "Пропустить"
- **Валидация**: Проверка данных на каждом шаге
- **Подсказки**: Справка и инструкции на каждом шаге
- **Сохранение прогресса**: Автоматическое сохранение между шагами

## Определение завершенности

Система считает настройку завершенной, если:
- Создан хотя бы один администратор
- Настроен Telegram Bot Token
- Настроено подключение к MikroTik
- Сохранены основные настройки безопасности

После завершения мастер не запускается автоматически при входе.
