# Требования к системе

## Системные требования

### Сервер
- ОС: Debian 12 (Bookworm) или Debian 13 (Trixie)
- Python: 3.11 или выше (входит в Debian 12+)
- SQLite: 3.x (встроен в Python, не требует отдельной установки)
- RAM: минимум 512MB (рекомендуется 1GB+)
- Диск: минимум 1GB свободного места (рекомендуется 2GB+)
- Сеть: доступ к интернету и MikroTik роутеру

### Минимальные требования для разработки
- ОС: Debian 12+ или совместимая
- Python 3.11+
- SQLite 3.x (обычно встроен в Python)
- Node.js 18+ (для frontend, если требуется сборка)
- Git (для клонирования репозитория)

## Зависимости

См. файлы `requirements.txt` и `package.json` (будут созданы при разработке)

## Доступы

### Для работы системы требуется:
- Telegram Bot Token
- Доступ к MikroTik роутеру по SSH
  - SSH ключ или логин/пароль
  - Права администратора на роутере

### Зависимости:
- Все зависимости включены в проект
- Python зависимости указаны в requirements.txt
- Frontend зависимости указаны в package.json
- Автоматическая установка через скрипт install.sh
