# Конфигурационные файлы

Эта папка содержит конфигурационные файлы проекта.

## Структура

- `config.example.yaml` - пример конфигурационного файла (будет создан при разработке)
- `settings.json` - файл настроек по умолчанию (будет создан при разработке)

## Примечание

Конфигурационные файлы с чувствительными данными (пароли, токены) должны быть исключены из системы контроля версий и храниться в переменных окружения или отдельном файле `.env`.
