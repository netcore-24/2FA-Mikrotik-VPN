# Скрипты

Эта папка содержит вспомогательные скрипты для управления проектом.

## Примеры скриптов (будут созданы при разработке)

- `setup.sh` - скрипт начальной настройки проекта
- `backup_db.sh` - скрипт резервного копирования базы данных
- `migrate.sh` - скрипт выполнения миграций
- `start.sh` - скрипт запуска приложения
- `stop.sh` - скрипт остановки приложения
